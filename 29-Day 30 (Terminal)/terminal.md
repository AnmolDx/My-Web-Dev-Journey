Command line - any interface that is used by entering textual commands
Terminal - This is a type of command line 
Console - A command-line interface used to work with your computer
Shell - A program running on terminal

Basic commands
ls - list files (show files in current directory)
pwd - print working directory (where am i?)
clear - clear the console 
up arrow key - to get last written command

Navigation Command
cd - change directory, use .. to go in parent folder
auto complete can be done by Tab key

Path 
Absolute path - with this we can access any directory from anywhere it path starts with /(forward slash)
Relative path - path can only be access if it is with the current directory start with (folder name)
/ = root directory
~ = home directory

Making Directories
mkdir - make directory (folders)

Flags - flags are characters that we pass with commands to modify their behaviour.
Manual command
man ls - give info about ls command if it is not working us ls --help
man mkdir - give info about mkdir command if it is not working us mkdir --help

Touch command - used to create files

Deleting Files & Folders - changes made by this is permanent make sure to use it wisely
rm - remove files
rmdir - removes empty folders
rm -rf(recursive, force) - remove folder forcefully
